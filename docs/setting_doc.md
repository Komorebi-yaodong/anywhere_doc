# 设置 ⚙️(Settings) 使用说明文档

**文档更新时间：2026年4月16日**

「设置」页面是 Anywhere Desktop 的全局控制中心。在这里，您可以对桌面行为、界面外观、全局快捷键、数据备份、WebDAV 同步以及语音功能进行精细配置。

合理的设置可以让桌面端更加贴合您的本机使用习惯。

---

## 1. 通用设置 (General Settings)

* **语言 (Language)**：切换界面语言。
* **主题 (Theme)**：
  * **跟随系统**：跟随 Windows / macOS 的深浅色模式；
  * **浅色 / 深色**：手动固定主题。
* **显示成功通知**：开启后，任务完成、保存成功等事件会给出提示。
* **固定窗口位置**：开启后，独立窗口会记忆位置，而不是每次重新按默认策略出现。

### 全局行为控制

这些设置可作为新建助手的默认值，或用于统一已有助手的行为：

* **全局窗口置顶**
* **全局失焦关闭**
* **全局自动保存对话**

### 输入习惯

* **发送时跳过换行符**：适合处理复制自 PDF / 网页的断行文本。
* **Ctrl+Enter 发送**：
  * 开启：`Enter` 换行，`Ctrl+Enter` 发送；
  * 关闭：`Enter` 发送，`Shift+Enter` 换行。

---

## 2. 桌面端专属设置 (Desktop)

这是 Anywhere Desktop 相比原插件最关键的一组设置。

### 2.1 全局快捷键

您可以配置桌面端的全局快捷键，例如：

* **主控台显示 / 隐藏**
* **快捷召唤窗口 (Quick Summon)**
* **自动追问 / 全局追问**
* **某个快捷助手的专属快捷键**

这些快捷键在系统任意软件中都可生效，但请避免与系统或常用软件热键冲突。

### 2.2 托盘与关闭行为

* **关闭到托盘**：关闭窗口时不退出程序，而是最小化到系统托盘。
* **托盘常驻**：适合需要全局快捷键、定时任务和后台 AI 能力长期运行的场景。

### 2.3 用户身份信息

桌面端支持设置：

* **用户名**
* **用户头像**

这些信息会在独立对话窗口中显示，使用户消息与 AI 消息更对称、更具“聊天应用”体验。

---

## 3. 语音设置 (Voice Settings)

Anywhere Desktop 支持语音输出，让 AI 可以“说话”。

* **语音列表管理**：添加、编辑、删除语音预设；
* **常见格式**：`voiceId` 或 `voiceId - note`；
* **使用方式**：在快捷助手配置中为特定助手指定默认语音。

---

## 4. 数据管理 (Data Management)

数据安全非常重要，桌面端支持完整的配置导入导出：

* **导出配置**：将快捷助手、服务商、MCP、部分 Skill 绑定关系等配置打包成 `.json` 文件；
* **导入配置**：从导出的 `.json` 中恢复配置。

> 注意：配置文件中可能包含 API Key，请妥善保管，不要外泄。

---

## 5. WebDAV 云同步 (Cloud Sync)

这是桌面端跨设备同步的核心能力。

### 5.1 配置项

* **WebDAV 地址**：如坚果云、Nextcloud、NAS、Koofr 等；
* **用户名 / 密码**：认证信息；
* **配置路径**：存放桌面端配置文件；
* **数据路径**：存放历史对话文件。

### 5.2 本地存储

* **本地对话路径**：建议务必配置，用于保存自动保存的 JSON 对话记录。

### 5.3 功能按钮

* **备份到 WebDAV**：手动备份当前配置；
* **管理备份**：查看并恢复云端历史备份。

## 6. 常见问题 (FAQ)

**Q1: 为什么修改了“全局窗口置顶”，有些助手没变化？**

> **A:** 某些助手可能已拥有独立配置。再次统一切换全局设置时，桌面端会尝试批量覆盖已有助手配置。

**Q2: WebDAV 连接失败怎么办？**

> **A:**
>
> 1. 检查 URL 是否带协议头；
> 2. 部分服务商需要应用专用密码；
> 3. 检查网络与代理环境是否能访问该服务。

**Q3: 为什么语音没有声音？**

> **A:**
>
> 1. 确认模型支持语音输出；
> 2. 确认语音 ID 正确；
> 3. 检查系统音量与输出设备。

**Q4: 为什么全局快捷键没有生效？**

> **A:**
>
> 1. 检查是否与系统热键冲突；
> 2. 检查应用是否仍在后台运行；
> 3. 修改快捷键后可尝试重启桌面端以重新注册。
