# 历史对话 💬(Chat History) 使用说明文档

**文档更新时间：2026年4月16日**

欢迎使用 **Anywhere Desktop** 的历史对话功能。本模块用于统一管理、备份与恢复您和 AI 的桌面端对话记录。无论是日常聊天、快捷助手多轮会话，还是定时任务自动生成的执行结果，都可以在这里集中查看。

桌面端延续了本地优先的数据策略，并支持通过 WebDAV 在多台电脑之间同步会话数据。

---

## 1. 核心概念

在使用历史对话功能前，请先理解以下三类数据视图：

### 📂 本地对话 (Local Chats)

* **定义**：存储在当前电脑磁盘中的对话记录。
* **存储机制**：
  1. **自动保存**：当会话开启自动保存后，桌面端会周期性将当前会话保存到本地目录。
  2. **任务生成**：定时任务执行完成后，也可自动生成对应的对话文件。
* **文件格式**：统一使用标准 `.json` 会话文件，包含消息、标题、时间、模型与必要的上下文信息。
* **依赖条件**：需要在「设置 -> 数据管理 / WebDAV」中配置本地对话路径。

### ☁️ 云端对话 (Cloud Chats)

* **定义**：存储在您配置的 WebDAV 服务器上的会话备份。
* **用途**：用于跨设备同步。例如，您在办公室电脑保存的桌面端对话，可在家中电脑下载后继续处理。
* **依赖条件**：需要在「设置」中正确配置 WebDAV 地址、账号与目录。

### 👌 恢复对话 (Resume Chats)

* **定义**：将已保存的 JSON 对话重新载入到桌面端独立窗口继续聊天。
* **用途**：用于继续先前会话、查看定时任务历史执行过程，或打开其他设备导出的会话文件。
* **依赖条件**：无。

---

## 2. 界面与操作指南

### 2.1 视图切换

在主控台的“历史对话”页面，您可以在 **本地对话** 与 **云端对话** 之间切换。

* 若未配置 WebDAV，云端视图将不可用。
* 若未配置本地对话目录，本地视图将为空，并提示您先完成路径配置。

### 2.2 数据同步

Anywhere Desktop 提供了面向本地 / 云端的双向同步能力：

* **📤 上传变更**：当本地会话比云端更新，或本地新增了会话文件时，可批量上传到 WebDAV。
* **📥 下载变更**：当云端存在新文件，或云端版本更新时，可批量拉取到本地。
* **单文件强制同步**：在文件列表中可按单项执行同步，用当前版本覆盖另一端。

### 2.3 文件管理

* **进入对话**：点击 **聊天 / 恢复** 按钮，可立即在桌面端独立窗口中打开该历史会话。
  * *场景*：查看昨晚定时任务生成的 AI 日报，并继续追问。
* **重命名**：支持对本地或云端会话文件重命名。
* **删除**：支持删除单个会话，并可按提示决定是否同步删除另一端副本。
* **批量操作**：支持勾选多个历史文件后统一删除。

### 2.4 自动清理

当历史记录过多时，可以使用右上角的清理能力：

1. 选择保留时间范围（如最近 3 天、7 天、30 天或自定义天数）。
2. 预览待删除文件数量与空间占用。
3. 确认后批量清理当前视图中的过期历史文件。

> 注意：自动清理默认只作用于当前视图，不会无提示地联动删除另一端数据。

## 3. 对话的保存与导出

在桌面端独立对话窗口中，您可以通过以下方式生成或导出历史记录：

1. **自动保存**：
   * **普通会话**：开启自动保存后，会话会周期性落盘到本地目录。
   * **定时任务**：开启任务自动保存后，执行完成时会自动生成会话文件。
2. **手动保存 / 另存为**：
   * 可通过窗口内保存入口，或使用快捷键 `Ctrl + S`。
   * **保存为 JSON**：用于在 Anywhere Desktop 中恢复并继续聊天。
   * **保存为 Markdown**：适合阅读、归档或发布到笔记 / 博客。
   * **保存为 HTML**：保留较完整的对话样式、头像与时间信息。
   * **保存到云端**：直接将当前对话上传至 WebDAV。

## 4. 常见问题 (FAQ)

**Q1: 为什么我看不到“本地对话”列表？**

> **A:** 请先在「设置」中配置有效的本地对话路径。建议选择一个专门用于保存会话的文件夹，例如 `D:\Anywhere_Chats`。

**Q2: 定时任务生成的对话文件名格式是什么？**

> **A:** 通常为 `任务名称-YYYYMMDD-HHMMSS.json`，例如：`每日早报-20260416-080000.json`。

**Q3: 我在 A 电脑上删除了一个对话，B 电脑上还会存在吗？**

> **A:** 如果您只删除了本地文件而没有同步删除云端版本，另一台电脑在同步时仍可能再次拉取该文件。建议在删除时留意同步提示。

**Q4: 自动保存的文件名是如何生成的（非定时任务）？**

> **A:** 系统会根据对话标题、首条内容摘要或时间信息生成默认名称，之后您仍可在历史对话页中手动重命名。

**Q5: 导入的 JSON 文件格式有什么要求？**

> **A:** 建议使用 Anywhere Desktop 导出的标准 JSON 会话文件。手工修改 JSON 可能造成字段不兼容或显示异常。

**Q6: 如果我重装了系统，历史记录会丢失吗？**

> **A:**
>
> * **本地对话**：如果会话目录放在非系统盘、NAS 同步盘或云盘目录中，重装后重新指定目录即可恢复。
> * **云端对话**：只要 WebDAV 服务中的数据仍在，重新配置 WebDAV 后即可找回云端历史。
