## 说明

1. 本仓库中的 `docs/*.md` 会被 Anywhere Desktop 主控台“使用指南”直接读取。
2. 文档内图片统一放在 `image/` 目录下，供桌面端在线帮助渲染使用。
3. 若更新文档内容，请同时保持文件名稳定，避免桌面端内置文档索引失效。

欢迎提交 Issue 或 PR 持续完善文档。
